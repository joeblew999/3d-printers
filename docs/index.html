x1ctl downloads

Latest binaries are attached to the most recent tagged release (releases/latest).

Download links

  • Linux amd64
  • Linux arm64
  • macOS amd64
  • macOS arm64 (Apple Silicon)
  • Windows amd64
  • Windows arm64

Usage

./x1ctl -ip 192.168.1.50 -access-code ABCD

Enable LAN Mode on the printer, note the IP and access code, and run the binary for your platform.

How releases are built

  • CI builds linux/windows/mac binaries for amd64 and arm64 on every tag (v*).
  • Pages is deployed from the docs/ folder via GitHub Actions.
